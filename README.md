# Small Equipment Tool Sell-Off (S.E.T.S)

A sophisticated React-based e-commerce application featuring real-time data filtering, internationalization, and enterprise-grade architecture.

## Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **State Management**: Zustand with persistence
- **UI Framework**: Chakra UI + Emotion
- **Data Fetching**: TanStack Query + Axios
- **Routing**: React Router DOM
- **Styling**: SCSS + Chakra UI
- **i18n**: i18next
- **Icons**: FontAwesome

## Key Features
- Dynamic data table with 50+ columns
- Advanced filtering (search, dropdowns, price range)
- Pagination with URL parameterization
- Multi-language support (EN/FR)
- WCAG 2.1 AODA AA compliance
- Responsive design
- Real-time search with debouncing
- Persistent state management

## Architecture Highlights
- Component-based architecture with feature separation
- Centralized state management with Zustand
- API caching with React Query (24hr persistence)
- Environment-specific configurations
- TypeScript interfaces for type safety
- Modular SCSS organization
